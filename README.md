# maksimovic/zend-soap

A PHP 8.1+ compatible fork of [zf1s/zend-soap](https://github.com/zf1s/zend-soap), originally from [Zend Framework 1](https://github.com/zendframework/zf1).

SOAP client and server with WSDL auto-discovery. Supports complex types, document/literal style, and multiple WSDL strategies.

## Installation

```bash
composer require maksimovic/zend-soap
```

This package replaces `zf1/zend-soap` and `zf1s/zend-soap`.
